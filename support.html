MIKHMON

FAQ

  1. Tanya : Apakah Mikhmon bisa digunakan untuk Userman?
    Jawab : Tidak.
  2. Tanya : Apakah Mikhmon gratis?
    Jawab : Iya.
  3. Tanya : Apakah ada layanan premium untuk Mikhmon?
    Jawab : Ada, silakan kunjungi https://mikhmon.online
  4. Tanya : Apakah Mikhmon bisa diinstall di Mikrotik?
    Jawab : Tidak, Mikhmon diinstall di PC Windows, Linux, atau modem dengan OS OpenWRT.
  5. Tanya : Apakah Mikhmon bisa dipasang di hosting?
    Jawab : Bisa, untuk tutorial silakan googling.
  6. Tanya : Apakah Mikhmon bisa digunakan untuk RB 941, RB 750?
    Jawab : Secara teori Mikhmon bisa digunakan diberbagai tipe Router Board, dengan syarat sudah menggunakan RouterOS versi 6.3 keatas. Disarankan RB yang digunakan memiliki clock speed CPU minimal 650 MHz.
  7. Tanya : Bagaimana cara install Mikhmon di PC atau di Android?
    Jawab : Panduan instalasi Mikhmon sudah disediakan di website, klik di sini atau di sini
  8. Tanya : Kenapa Mikhmon tidak bisa konek ke Mikrotik?
    Jawab :
    1. Pastikan Mikhmon sudah satu jaringan dengan Mikrotik, lebih baik lagi satu ip class.
    2. Pastikan port API Mikrotik sudah aktif dengan port 8728.
    3. Pastikan username dan password Mikrotik yang dimasukkan di settings Mikhmon sudah benar.
    4. Pastikan tidak dihalangi di firewall.
  9. Tanya : Apakah Time Limit harus diisi saat generate user?
    Jawab : Tidak harus.
    1. Time Limit, Data Limit, Prefix adalah pilihan yang tidak diharuskan.
    2. Time limit diisi juka ingin membuat User Hotspot dengan 2 batasan waktu. Sebagai contoh, User dengan batas pemakaian 5 jam dan masa aktif 1 hari. Maka yang harus dibuat adalah User Profile dengan Validity 1d, kemudian buat User Hotspot dengan Time Limit 5h.
  10. Tanya : Grace period itu apa?
    Jawab : Grace period adalah masa tenggang sebelum User Hotspot benar-benar dihapus. User Hotspot tidak akan bisa login selama masa tenggang tersebut. Tujuannya agar pelanggan tahu bahwa User Hotspotnya sudah tidak bisa digunakan lagi. Grace period aktif setelah Validity habis.
  11. Tanya : Apakah Time Limit harus diisi saat Add User atau Generate User?
    Jawab : Tidak harus. Pembuatan User Hotspot dengan Time Limit *Time Limit lebih kecil dari Validity. Contoh: Durasi 5jam masa aktif 1hari. Generate user dengan time limit 5h, kemudian pilih Profile dengan validity 1d.
  12. Tanya : Apakah bisa menghapus User Hotspot lebih dari satu?
    Jawab : Bisa, menghapusnya berdasarkan kode generate. Masuk ke menu User List, klik pada kode generate yang ingin dihapus.
  13. Tanya : Bagaimana cara cetak ulang User Hotspot yang sudah pernah di generate?
    Jawab : Masuk menu User List, klik pada kolom print sesuai dengan kode generate yang ingin dicetak ulang.
  14. Tanya : Kenapa user / voucher baru dipakai sudah expired?
    Jawab :
    1. Pastikan sudah setting jam dan tanggal dengan benar. (Winbox : System -> Clock).
    2. Pastikan memilih zona waktu sesuai daerah Anda. Hilangkan centang pada “Time Zone Autodetect”.
    3. Tambahakan IP NTP Server pada SNTP Client (Winbox : System -> SNTP Client).
      1. Primary NTP Server : 114.141.48.158
      2. Secondary NTP Server : 202.162.32.12
  15. Tanya : Bagaimana cara mengaktifkan fitur Selling Report?
    Jawab : Buat User Profile dengan expired mode Remove & Record atau Notice & Record. Penjelasan lebih lengkap klik di sinni.
  16. Tanya : Bagaimana cara menghapus Selling Report / User Log?
    Jawab : Masuk ke menu Report kemudian filter/klik pada tanggal, ( contoh berdasarkan bulan klik pada jan/, berdasarkan hari klik pada 01/2018).
  17. Tanya : Apakah User Profile denga Shared Users lebih dari satu bisa menggunakan Lock User?
    Jawab : Tidak, jika menggunakan Lock User maka Shared Users tidak berfungsi.
  18. Tanya : Bagaimana cara mengganti logo voucher?
    Jawab :
    1. Masuk menu Settings -> Upload Logo.
    2. Pastikan logo yang diupload menggunakan format “.png" dan nama file sesuai yang muncul pada form upload logo dan ukuran file tidak lebih dari 1MB.
    3. Setelah upload berhasil, bersihkan cache browser.
    4. Atau cek videonya, klik di sini.
  19. Tanya : Bagaimana cara mengganti template voucher Mikhmon?
    Jawab :
    1. Untuk mengganti template voucher Mikhmon harus memahami HTML dan PHP.
    2. Jika tidak ingin repot coding atau tidak bisa coding, bisa menggunakan template yang disediakan di website Mikhmon atau di dokumen Grup.
    3. Untuk Mengganti template, masuk menu Setting -> Template Editor, edit atau paste template Default kemudian save.
  20. Tanya : Kenapa template vouchernya tidak berubah, padahal sudah ganti di template editor?
    Jawab : Template voucher mikhmon ada 3 jenis, default, thermal dan small. Jika print 1 voucher maka yang dipakai adalah template thermal. Jika prin > 1 maka template yang dipakai bisa pilih default atau small. Ketiga template itu berbeda model, template yang tersedia di website adalah template default.
