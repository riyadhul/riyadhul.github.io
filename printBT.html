Print BT server Termux

Set URL Printing Access

URL Printing Access

  1. Print default : http://ip-mikhmon-server/bt/?key=your-btkey
  2. Print with logo : http://ip-mikhmon-server/bt/?key=your-btkey&logo
  3. Print with QR : http://ip-mikhmon-server/bt/?key=your-btkey&qr
  4. Print with logo and QR : http://ip-mikhmon-server/bt/?key=your-btkey&logo&qr

URL Printing Access [HTTPS]

  1. Print default : https://your-domain/bt/?key=your-btkey&https
  2. Print with logo : https://your-domain/bt/?key=your-btkey&logo&https
  3. Print with QR : https://your-domain/bt/?key=your-btkey&qr&https
  4. Print with logo and QR : https://your-domain/bt/?key=your-btkey&logo&qr&https

Untuk print BT dengan bahasa Inggris tambahkan &en dibagian akhir URL
Contoh : http://ip-mikhmon-server/bt/?key=your-btkey&logo&qr&en
