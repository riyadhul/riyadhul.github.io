MIKHMON

Template Hotspot Mikhmon

Template berikut menggunakan dua cara login yaitu, login voucher dan user password.
Login Voucher menggunakan username yang dibuat sama dengan password sedangkan user password, username berbeda dengan password.

Lisensi

Template ini gratis, silahkan edit yang perlu disesuaikan. Jangan menghilangkan atau merubah bagian footer (Powered by Mikhmon)!

Yang Perlu Disesuaikan

  • Daftar paket wifi disesuaikan di file (paket-wifi.csv) dengan text-editor (notepad atau notepad++).
  • Logo dapat diganti dengan format (logo.png dan sponsor.png) ditaruh di folder img.
  • hotspot2
    1. Sesuaikan juga yang dirasa perlu disesuaikan (nama usaha alamat dll) di login.html.
    2. * hotspot2 tidak memiliki notifikasi expired dan lainnya seperti di hotspot3 dan hospot4.
  • hotspot3, hotspot4
    1. Nama usaha, alamat, notifkasi expired dan lainnya di (data.js).
    2. Template tidak menggunakan notifikasi expired di laman terpisah, melainkan menggunakan modal sudah terintegrasi di login.html.
    3. Notifikasi Expired bereaksi terhadap limit uptime, untuk lebih mudah bisa menggunakan MIKHMON. Dengan user profile yang menggunakan Expired Mode "Remove/Remove+Record".
  • Diperlukan webserver Mikhmon untuk menampilkan informasi expired di laman status hotspot, untuk panduan menampilkan status expired klik di sini.
